# MocDoc Report Automation

This project automates the process of downloading daily reports from MocDoc.

## Features
- Downloads Current Stock (Batchwise)
- Downloads Daily Sales (Bill Wise Detailed)
- Downloads Daily Purchase (GRN Detailed)
- Automatically sets correct dates
- Handles sync and processing states
- Saves reports as CSV files

## Tech Used
- Python
- Playwright

## Use Case
Designed to eliminate manual report downloading in pharmacy operations.

## Status
Working automation script used for daily operations.
